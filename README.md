## Install
```
npm i waterline-auto -g
```

## Usage
type `wa` | `waterline-auto`

```shell
$ wa [-h host] [-p port] [-u user] [-a password] [-d database] [-t tables]
```

### Notes
- `-a` take from `redis-cli`, `authentication`
